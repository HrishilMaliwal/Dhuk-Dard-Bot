Our model is trained on a dataset that contains real patient-therapists interactions

Trained using Logistic Regression Classifier

The idea behind Therap.ai is to use a Logistic Regression Classifier based training model due to certain superior features like accepting the entire input sequence and lower complexity of the self-attention mechanism. The transformer architecture comprises an encoder-decoder system where the input from the user is provided as an input to the encoder. The transformer model has also shown promising results in several previous studies.
